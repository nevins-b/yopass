Yopass

Yopass Keep a secret

Protected by state of the art cryptography

Encryption and decryption is being made locally in your browser. The decryption key is never stored online

In memory storage

The encrypted message is never stored on disk leaving no traces of your data.

Self destruction

All messages have a fixed time to live and will be deleted automatically after expiration.

Only downloadable once

The encrypted message can only be downloaded once before automatic deletion.

Extend and embed

This site is just a frontend to the Yopass API, use it to extend other applications.

Open Source Software

Yopass is an Open Source project meaning full transparency and the possibility to submit features, fix bugs or run the software yourself
